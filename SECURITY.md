# Security Policy

## Supported Versions

| Version | Supported          |
|---------| ------------------ |
| 0.9.x   | :white_check_mark: |
| < 0.9.0 | :x:                |

## Reporting a Vulnerability

If you have discovered a security issue, please report it by creating an [issue](https://github.com/webdiscus/html-bundler-webpack-plugin/issues).

Thanks for helping.
